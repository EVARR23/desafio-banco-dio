# Sistema Bancário em Python

Um sistema simples de banco desenvolvido em Python para gerenciar contas, depósitos, saques e transferências.

## Funcionalidades

- Criar contas bancárias
- Consultar saldo
- Depositar dinheiro
- Sacar dinheiro
- Transferir entre contas

## Requisitos

- Python 3.x instalado
